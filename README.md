# Python Assignment — Part 3: File I/O, APIs & Exception Handling

## Theme: Product Explorer & Error-Resilient Logger

A program that fetches real product data from a public API, processes it, saves results to files, and handles all failure scenarios gracefully.

## Tasks

- **Task 1** — File Read & Write Basics (6 marks)
- **Task 2** — API Integration using DummyJSON (8 marks)
- **Task 3** — Exception Handling (7 marks)
- **Task 4** — Logging to File (4 marks)

## How to Run

Install the required library first:
```
pip install requests
```

Then run:
```
python part3_api_files.py
```

> Requires an internet connection for API calls.
> The script will ask for a keyword input (Task 1) and product ID lookups (Task 3D) — follow the prompts.
